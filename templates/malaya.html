Malaya OS 1.0

Malaya OS is a hobby Linux distro built completely from scratch. It was initially based on LFS (Linux From Scratch). A lot of time and research went into it since some things have to be figured out on my own and aren't documented in LFS. And this is not AI-generated by the way! 😀

You can download the ISO file (a live OS with an installer) below to try it. If you plan to install it on your system, make sure Secure Boot is turned off and that you are using UEFI mode. It may still boot using legacy mode but you might encounter errors during installation. You need at least 20GB of free storage space.
